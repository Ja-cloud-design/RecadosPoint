Instrucciones

1. Conceder los permisos

Recados Point necesita ubicación precisa, ubicación permanente y notificaciones para detectar una ubicación cuando la aplicación no está abierta.

  1. Abre Recados Point → Permisos.
  2. Al activar o desactivar Ubicación precisa, la app mostrará la ruta: Permisos → Ubicación → Usar ubicación precisa.
  3. Al activar o desactivar Ubicación permanente, la app mostrará la ruta: Permisos → Ubicación → Permitir siempre.
  4. Al activar o desactivar Notificaciones, la app mostrará la ruta: Notificaciones → Permitir notificaciones.
  5. En Android 14 o superior, activa también Alarmas a pantalla completa para que la alarma y el aviso por voz ocupen toda la pantalla.
En teléfonos Samsung, si el seguimiento se detiene al dejar la aplicación en segundo plano, añade Recados Point a Ajustes → Mantenimiento y batería → Batería → Límites de uso en segundo plano → Aplicaciones sin suspensión.

2. Crear una ubicación

  1. Pulsa el botón +.
  2. Escribe el nombre de la ubicación y, si lo necesitas, datos adicionales.
  3. Busca una dirección, utiliza el botón de diana o mantén pulsado un punto del mapa.
  4. Selecciona el radio exacto, el tipo de aviso, los días y el horario.
  5. Si eliges Aviso por voz, puedes escribir un mensaje personalizado.
  6. Pulsa Guardar. Se abrirán los recados de esa ubicación.
Los recados no son obligatorios. Si una ubicación está activa, puede avisar aunque no tenga recados pendientes.

3. Días activos y aviso de una sola vez

Los días seleccionados indican cuándo puede repetirse una ubicación.

  • Con días seleccionados: la ubicación puede avisar esos días, respetando el horario configurado.
  • Sin ningún día seleccionado: la ubicación avisa una sola vez. Después del aviso se desactiva hasta que la vuelvas a activar manualmente.

Esta lógica funciona como una alarma normal: sin días marcados es un aviso único; con días marcados es un aviso repetitivo según esos días.

4. Añadir, completar y ordenar recados

Añade los recados que quieras recordar al llegar a la ubicación. Cuando se abre el popup, el campo de texto recibe el foco y se muestra el teclado para escribir directamente.

  • Añadir: guarda el recado y cierra el popup.
  • Siguiente: guarda el recado y deja el popup preparado para escribir otro.
  • Tocar el texto: tacha o desmarca el recado.
  • Mantener pulsado y arrastrar: permite cambiar el orden manual de los recados.
  • Marcar/desmarcar todos: alterna entre tachar todos los recados o dejarlos todos pendientes.

5. Funcionamiento de las ubicaciones

Una ubicación puede vigilarse cuando se cumplen estas condiciones:

  • Está activada manualmente.
  • Está dentro del radio configurado.
  • Está dentro del horario configurado, si tiene horario.
  • El día actual está permitido, o no hay ningún día seleccionado porque es un aviso de una sola vez.
  • No ha generado ya el aviso que le corresponde.

Si la ubicación no tiene recados, puede avisar igualmente. El aviso se basa en la ubicación activa, no en que existan recados pendientes.

6. Estados

  • Activa: puede detectar la entrada y avisar.
  • En espera · Fuera de horario: está activada, pero el horario no permite vigilarla en este momento.
  • Desactivada para hoy: el día actual no está seleccionado.
  • Aviso único: no tiene días seleccionados y avisará una sola vez.
  • Avisado hoy: ya generó un aviso durante el día actual.
  • Desactivada manualmente: el usuario apagó la ubicación.

7. Tipos de aviso

  • Notificación: muestra un aviso normal con el nombre de la ubicación. El sonido y la vibración se gestionan desde los ajustes de notificaciones de Android.
  • Alarma: abre una alarma a pantalla completa con sonido repetitivo. Permite aplazarla el tiempo configurado o detenerla.
  • Aviso por voz: anuncia el mensaje automático o personalizado, respetando el descanso configurado entre repeticiones. Permite aplazarlo o detener la alarma.
Privacidad: la pantalla de alarma y las notificaciones no muestran la lista de recados. Solo se muestra la información necesaria para identificar el aviso.

En Configuración puedes ajustar el tiempo máximo de alarma y voz, el descanso entre repeticiones de voz, el tiempo de aplazamiento y el sonido de alarma. Si no realizas ninguna acción, el sonido o la voz se detienen y queda una notificación.

Si el motor de voz no está disponible, la aplicación cambia a alarma normal para evitar que el aviso falle.

8. Buscar, ordenar y eliminar ubicaciones

Utiliza el buscador para localizar una ubicación. El icono de ordenación permite ordenar por nombre, fecha de creación u orden manual.

  • Pulsación rápida: abre los recados.
  • Pulsación larga y soltar: selecciona una o varias ubicaciones para eliminarlas.
  • Pulsación larga y arrastrar: cambia el orden manual.

9. Configuración

  • Tiempo máximo de alarma y voz: permite indicar segundos o minutos mediante slider.
  • Descanso entre repeticiones de voz: añade una pausa entre repetición y repetición para que no resulte tan continuo.
  • Aplazar alarma: permite elegir el tiempo del botón de aplazamiento, por ejemplo 5, 10 o 15 minutos.
  • Días activos predeterminados: pueden quedar todos desmarcados para crear avisos de una sola vez.
  • Exportar configuración e importar configuración: permiten guardar o restaurar los ajustes de la aplicación.

10. Copias de seguridad

Exportar ubicaciones abre el selector de Android para elegir la carpeta y modificar el nombre del ZIP. El nombre propuesto empieza por RecadosPoint_ubicaciones_.

Al usar Importar ubicaciones puedes fusionar la copia con los datos actuales o reemplazar todos los datos.
